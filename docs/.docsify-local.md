# Testing Docsify Locally

To test the documentation site locally:

```bash
# Install docsify-cli globally (one time)
npm i -g docsify-cli

# Serve the docs
cd docs
docsify serve .

# Open browser to http://localhost:3000
```

Alternatively, use Python's built-in server:

```bash
cd docs
python3 -m http.server 3000

# Open browser to http://localhost:3000
```
